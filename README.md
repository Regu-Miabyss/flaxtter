A third-party Twitter/X client for **Linux** and **Android**, built with Flutter. Session login uses a WebView; the API layer is adapted from [Squawker](https://github.com/j-fbriere/squawker).

This is not an official X product. Use it at your own risk.

## Highlights

- **No trackers** — no third-party analytics or telemetry; your session stays on your device
- **Save tweets as images** — export a tweet card as a PNG, save to your gallery, and copy to the clipboard on Linux
- **Customizable look** — import your own font (TTF/OTF), choose accent colors or Material You dynamic color, adjust text size, and switch light/dark/system theme
